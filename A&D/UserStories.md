__Admin:__

As an admin, I want to log in into the system, so that I can access all the functionality the system provides. <br>
As an admin, I want to create new tasks by selecting building blocks or using templates, so that I can quickly set up event functionality. <br>
As an admin, I want to view the leaderboard, so that I can monitor team progress and standings during the event. <br>
As an admin, I want to delete tasks, so that I can remove any incorrect or outdated tasks from the system. <br>
As an admin, I want to generate QR codes, so that participants can use them to register as team leaders or access tasks during the event. <br>

__Teamleader:__

As a teamleader, I want to create a new team via a QR code, so that I can participate in the event. <br>
As a teamleader, I want to join an existing team by scanning a QR code, so that I can participate in an ongoing team effort. <br>
As a teamleader, I want to access my teams page, in order to see the score, placement and already completed tasks of my team. <br>
As a teamleader, I want to scan QR codes at checkpoints to access tasks and submit results, so that I can earn points for my team. <br>
As a teamleader, I want to see my team's progress (today and overall) and compare it to other teams' scores, so that I can stay informed and motivated. <br>
As a teamleader, I want to be able to leave my team, in case a new teamleader is needed. <br>
